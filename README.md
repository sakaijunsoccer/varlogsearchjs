# VarLogSearchJs

## QuickStart

#### Install node, npm
```
$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
$ nvm --version
0.39.3
$ nvm install 18.12.1
$ npm --version
8.19.2
```

#### Install package
```
$ npm install
```

#### Test
```
npm test
```

#### Run benchmark
```
npx ts-node tools/benchmark.ts
```
